# Self-Discipline（自我管理）

## 背景

自己的自律能力太差了，想找一个能够设置任务和闹钟提醒的app，但是在应用商城没找到自己喜欢的，于是就想自己做一个

## 功能

1. 设置任务

   * 任务分为四类：已完成任务、每日任务、今日任务、长期任务
